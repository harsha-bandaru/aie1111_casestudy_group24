## Problem Description
Manual courier tracking often leads to delays, lack of status visibility, and inaccuracies in record-keeping. This project implements a centralized digital system to manage tracking IDs, update delivery statuses in real-time, and maintain reliable logs for admins and customers.

## How to Run the Code
1. Ensure you have the **Java Development Kit (JDK)** installed.
2. Download the `CourierApp.java` file from the `/src` folder.
3. Open your terminal or command prompt.
4. Compile the code: `javac CourierApp.java`
5. Run the application: `java CourierApp`
